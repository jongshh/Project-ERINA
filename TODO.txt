L.T.M (Long-Term Memory Module)

1. data/Long-Term-Memory.Json에 각각의 특징들을 나열하는 어레이를 생성. (외형, 성격, 아는 사람, 좋아하는 것, 싫어하는 것...)
2. 대화가 충분히 (15번 정도) 이루어지면, L.T.M 생성 모델이 해당 대화록 (S.T.M, Short-Term-Memory) 에서 15번 동안의 대화동안 추출할 수 있는 외형, 성격, 아는 사람.. 등등을 읽고, 정해진 프롬프트 (L.T.M.py)의 형식대로 내보냄.
3. 내보내진 데이터는, 각각의 어레이에 추가되고, 이 과정에서 중복되지 않도록 검사를 실행하여 추가. 
4. 이렇게 저장된 데이터는, ERINA 모델이 대화를 생성할 때 참고할 수 있도록 함.
반복.
